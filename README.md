## Install

```bash
npm i -g kryptorious-portcheck
```

## Quickstart

```bash
npm i -g kryptorious-portcheck
portcheck 3000
```

## Commands

| Command | Description |
|---------|-------------|
| `portcheck 3000` | Check if a port is in use. |
| `portcheck scan 3000-3010` | Scan a range for conflicts. |
